NAME
-------------------------------------------------

::

          pwdx - report current working directory of a process


---------------------------------------------------------

::

          pwdx [options] pid [...]


-------------------------------------------------------

::

          -V, --version
                 Output version information and exit.

          -h, --help
                 Output help screen and exit.


---------------------------------------------------------

::

          ps(1), pgrep(1)


-----------------------------------------------------------

::

          No standards apply, but pwdx looks an awful lot like a SunOS
          command.


-----------------------------------------------------

::
