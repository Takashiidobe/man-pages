NAME
-------------------------------------------------

::

          dh_installmime - install mime files into package build
          directories


---------------------------------------------------------

::

          dh_installmime [debhelper options]


---------------------------------------------------------------

::

          dh_installmime is a debhelper program that is responsible for
          installing mime files into package build directories.


---------------------------------------------------

::

          debian/package.mime
              Installed into usr/lib/mime/packages/package in the package
              build directory.

          debian/package.sharedmimeinfo
              Installed into /usr/share/mime/packages/package.xml in the
              package build directory.


---------------------------------------------------------

          debhelper(7)

          This program is a part of debhelper.
