NAME
-------------------------------------------------

::

          xminicom - friendly serial communication program


---------------------------------------------------------

::

          xminicom minicom-options


---------------------------------------------------------------

::

          Xminicom is a script wrapper around minicom.  It tries to find a
          color capable xterm or rxvt on your system, and then runs minicom
          with the -c (color) flag in a terminal session.


-------------------------------------------------------

::

          See minicom (1).
