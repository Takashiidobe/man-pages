NAME
-------------------------------------------------

::

          isinf — test for infinity


---------------------------------------------------------

::

          #include <math.h>

          int isinf(real-floating x);


---------------------------------------------------------------

::

          The functionality described on this reference page is aligned
          with the ISO C standard. Any conflict between the requirements
          described here and the ISO C standard is unintentional. This
          volume of POSIX.1‐2017 defers to the ISO C standard.

          The isinf() macro shall determine whether its argument value is
          an infinity (positive or negative). First, an argument
          represented in a format wider than its semantic type is converted
          to its semantic type. Then determination is based on the type of
          the argument.


-----------------------------------------------------------------

::

          The isinf() macro shall return a non-zero value if and only if
          its argument has an infinite value.


-----------------------------------------------------
