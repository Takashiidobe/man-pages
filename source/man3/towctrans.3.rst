NAME
-------------------------------------------------

::

          towctrans - wide-character transliteration


---------------------------------------------------------

::

          #include <wctype.h>

          wint_t towctrans(wint_t wc, wctrans_t desc);


---------------------------------------------------------------

::

          If wc is a wide character, the towctrans() function translates it
          according to the transliteration descriptor desc.  If wc is WEOF,
          WEOF is returned.

          desc must be a transliteration descriptor returned by the
          wctrans(3) function.


-----------------------------------------------------------------

::

          The towctrans() function returns the translated wide character,
          or WEOF if wc is WEOF.


-------------------------------------------------------------

::

          For an explanation of the terms used in this section, see
          attributes(7).

-------------------------------------------------------------------

::

          POSIX.1-2001, POSIX.1-2008, C99.


---------------------------------------------------

::

          The behavior of towctrans() depends on the LC_CTYPE category of
          the current locale.
