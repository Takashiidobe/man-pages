NAME
-------------------------------------------------

::

          uuid_compare - compare whether two UUIDs are the same


---------------------------------------------------------

::

          #include <uuid.h>

          int uuid_compare(uuid_t uu1, uuid_t uu2)


---------------------------------------------------------------

::

          The uuid_compare() function compares the two supplied uuid
          variables uu1 and uu2 to each other.


-----------------------------------------------------------------

::

          Returns an integer less than, equal to, or greater than zero if
          uu1 is found, respectively, to be lexicographically less than,
          equal, or greater than uu2.


-------------------------------------------------------
