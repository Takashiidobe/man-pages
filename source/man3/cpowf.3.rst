NAME
-------------------------------------------------

::

          cpow, cpowf, cpowl - complex power function


---------------------------------------------------------

::

          #include <complex.h>

          double complex cpow(double complex x, double complex z);
          float complex cpowf(float complex x, float complex z);
          long double complex cpowl(long double complex x,
                                    long double complex z);

          Link with -lm.


---------------------------------------------------------------

::

          These functions calculate x raised to the power z (with a branch
          cut for x along the negative real axis.)


---------------------------------------------------------

::

          These functions first appeared in glibc in version 2.1.


-------------------------------------------------------------

::

          For an explanation of the terms used in this section, see
          attributes(7).

-------------------------------------------------------------------

::
