NAME
-------------------------------------------------

::

          gnutls_pubkey_get_pk_algorithm - API function


---------------------------------------------------------

::

          #include <gnutls/abstract.h>

          int gnutls_pubkey_get_pk_algorithm(gnutls_pubkey_t key, unsigned
          int * bits);


-----------------------------------------------------------

::

          gnutls_pubkey_t key
                      should contain a gnutls_pubkey_t type

          unsigned int * bits
                      If set will return the number of bits of the
                      parameters (may be NULL)


---------------------------------------------------------------

::

          This function will return the public key algorithm of a public
          key and if possible will return a number of bits that indicates
          the security parameter of the key.


-------------------------------------------------------

::

          a member of the gnutls_pk_algorithm_t enumeration on success, or
          a negative error code on error.
