NAME
-------------------------------------------------

::

          gnutls_pubkey_export_ecc_x962 - API function


---------------------------------------------------------

::

          #include <gnutls/abstract.h>

          int gnutls_pubkey_export_ecc_x962(gnutls_pubkey_t key,
          gnutls_datum_t * parameters, gnutls_datum_t * ecpoint);


-----------------------------------------------------------

::

          gnutls_pubkey_t key
                      Holds the public key

          gnutls_datum_t * parameters
                      DER encoding of an ANSI X9.62 parameters

          gnutls_datum_t * ecpoint
                      DER encoding of ANSI X9.62 ECPoint


---------------------------------------------------------------

::

          This function will export the ECC public key's parameters found
          in the given certificate.  The new parameters will be allocated
          using gnutls_malloc() and will be stored in the appropriate
          datum.


-------------------------------------------------------

::

          GNUTLS_E_SUCCESS on success, otherwise a negative error code.
