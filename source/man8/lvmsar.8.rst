NAME
-------------------------------------------------

::

          lvmsar — LVM system activity reporter


---------------------------------------------------------

::

          lvmsar


---------------------------------------------------------------

::

          lvmsar is not supported under LVM2. The device-mapper statistics
          facility provides similar performance metrics using the
          dmstats(8) command.


---------------------------------------------------------

::

          dmstats(8), lvm(8)
