NAME
-------------------------------------------------

::

          pam_selinux_check - login program to test pam_selinux.so


---------------------------------------------------------

::

          pam_selinux_check [user]


---------------------------------------------------------------

::

          With no arguments, pam_selinux_check will prompt for user


-------------------------------------------------------

::

          target_user
                 The user to login as.


---------------------------------------------------------------

::

          You must setup a /etc/pam.d/pam_selinux_check file, in order for
          the check to work.

          When checking if a selinux is valid, pam_selinux_check returns an
          exit code of 0 for success and > 0 on error:

          1: Authentication failure
