NAME
-------------------------------------------------

::

          xqmstats - Display XFS quota manager statistics from /proc


---------------------------------------------------------

::

          /usr/sbin/xqmstats


---------------------------------------------------------------

::

          xqmstat queries the kernel for the XFS Quota Manager dquot
          statistics.  It displays:

              •      Reclaims
              •      Missed reclaims
              •      Dquot dups
              •      Cache misses
              •      Cache hits
              •      Dquot wants
              •      Shake reclaims
              •      Inact reclaims


-------------------------------------------------------

::

          None.


---------------------------------------------------------

::

          quotastats(1), quota(1).
