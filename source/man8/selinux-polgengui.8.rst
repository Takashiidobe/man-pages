NAME
-------------------------------------------------

::

          selinux-polgengui - SELinux Policy Generation Tool


---------------------------------------------------------

::

          selinux-polgengui


---------------------------------------------------------------

::

          selinux-polgengui is a graphical tool, which can be used to
          create a framework for building SELinux Policy.


-------------------------------------------------------

::

          None


---------------------------------------------------

::

          /usr/bin/selinux-polgengui


---------------------------------------------------------

::

          To run the program type:

          selinux-polgengui


---------------------------------------------------------

::

          selinux(1),
                 sepolicy(8), sepolicy-generate(8)
